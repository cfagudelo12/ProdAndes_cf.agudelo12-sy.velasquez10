Para usar la aplicación se debe usar JBOSS 4.4.3. 
Luego de ejecutar JBOSS los pasos a seguir son los siguientes:
1. Se debe cambiar el archivo buidl.xml que se encuetra en la carpeta bin. Se cambia la linea 23: <property name="jboss.home" value="AQUI SE INDICA LA DIRECCION DEL JBOSS EN LA MAQUINA QUE SE ESTE EJECUTANDO" />.
2. Se corre con ant el archivo buidl.xml.
3. En un navegador como Google Chrome se escribe: localHost:8080/ProdAndes.
